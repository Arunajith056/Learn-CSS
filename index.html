#1.what is css

Introduction to CSS

- CSS stands for 'Cascading Style Sheets'

- It is Used to style and arrange our HTML elements like change the color,change the size and more and more we can do.

Why use Css

- CSS and Html work hand in hand

- Without CSS your website would not look appealing at all

- The latest Version Comes with Cool features Such as animation and transitions

What you Should Know

- You Should Know basic HTML as a minimum

- That,s all,remember in css we are styling and arranging our HTML elements.

#2.How to link HTML to CSS

1.External style sheet

- Create a file with the extension .css, for example, style.css, in your project folder.

- In the <head> section of your HTML document, add a <link> element to specify the location of your CSS file.

                        
    <!DOCTYPE html>
    <html lang="en">
    <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Your Title</title>
    <!-- Link to external CSS file -->
    <link rel="stylesheet" href="style.css">
    <!-- Explanation: The above line links your HTML document to an external CSS file named "style.css" located in your project folder. -->
    </head>
    <body></body>
    </html>
